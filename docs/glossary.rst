Glossary
========

.. glossary::

   analytical

       An analytical method is one that uses the rules of algebra and
       calculus to write down the solution as an exact formula without
       using estimation or approximation. An analytical solution is a
       solution that has been obtained using an analytical method. For
       example, you can write down the roots of a quadratic equation
       analytically using the quadratic formula (in contrast to
       obtaining them by a numerical root finding method).
